# Interview Answers

Be prepared to demonstrate your understanding of this week's concepts by answering questions on the following topics. These will not be counted as a part of your sprint score but will be helpful for preparing you for your endorsement interview, and enhancing overall understanding.

1. Explain what a token is used for.
   A token is used for the client to prove to the server it has the right to access certain information

2. What steps can you take in your web apps to keep your data secure?
   private routes with authentication

3. Describe how web servers work.
   Web servers receive HTTP Requests and respond with the correct data/web pages

4. Which HTTP methods can be mapped to the CRUD acronym that we use when interfacing with APIs/Servers.
   C - Create - POST
   R - Read - GET
   U - Update - PUT
   D - Delete - DELETE
